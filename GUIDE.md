# Light Chat 新功能演示指南

## 📅 会话日期显示

**位置**: 左侧边栏会话列表

每个会话现在会显示：
```
会话标题
6月6日 · GPT-4
```

## ✏️ 编辑用户消息

**如何使用**:
1. 将鼠标悬停在你发送的消息上
2. 点击左侧出现的铅笔图标
3. 修改消息内容
4. 按 `Enter` 发送，或 `Esc` 取消
5. 编辑后会删除该消息之后的所有对话，并用新内容重新开始

**应用场景**:
- 发现输入错误需要修正
- 想调整提问方式获得更好回答
- 需要补充更多上下文信息

## 🔄 重新生成 AI 回复

**位置**: AI 回复消息底部

**如何使用**:
1. 在 AI 回复下方找到"重新生成"按钮
2. 点击按钮
3. AI 会用相同的问题重新生成回答

**应用场景**:
- AI 回答不够满意
- 想要获得不同角度的答案
- 回答中断或不完整

## 📋 代码复制功能

**特性**:
- 代码块顶部显示语言类型（如 `javascript`、`python`）
- 右上角"复制"按钮一键复制代码
- 复制后显示"已复制"提示

## 📊 改进的表格渲染

**特性**:
- 圆角边框设计
- 表头深色背景突出显示
- 鼠标悬停行高亮
- 表格过宽时支持横向滚动

## 💡 使用技巧

### 编辑消息的最佳实践
1. 编辑后的对话会形成新的分支
2. 原有的后续对话会被删除
3. 适合在对话初期调整方向

### 重新生成的使用时机
- AI 回答太简短
- 需要更详细的解释
- 想要不同的解决方案
- 回答偏离了问题核心

## 🎯 快速上手

1. **发起对话** → 输入消息，选择模型
2. **查看历史** → 左侧边栏，带日期显示
3. **修正提问** → 悬停消息 → 点击编辑
4. **优化回答** → 点击"重新生成"按钮
5. **使用代码** → 点击代码块的"复制"按钮
