# Light Chat 新功能演示指南

## 📅 会话日期显示

**位置**: 左侧边栏会话列表

每个会话现在会显示：
```
会话标题
6月6日 · GPT-4
```

## ✏️ 编辑用户消息

**如何使用**:
1. 将鼠标悬停在你发送的消息上
2. 点击左侧出现的铅笔图标
3. 修改消息内容
4. 按 `Enter` 发送，或 `Esc` 取消
5. 编辑后会删除该消息之后的所有对话，并用新内容重新开始

**应用场景**:
- 发现输入错误需要修正
- 想调整提问方式获得更好回答
- 需要补充更多上下文信息

## 🔄 重新生成 AI 回复

**位置**: AI 回复消息底部

**如何使用**:
1. 在 AI 回复下方找到"重新生成"按钮
2. 点击按钮
3. AI 会用相同的问题重新生成回答

**应用场景**:
- AI 回答不够满意
- 想要获得不同角度的答案
- 回答中断或不完整

## 📋 代码复制功能

**特性**:
- 代码块顶部显示语言类型（如 `javascript`、`python`）
- 右上角"复制"按钮一键复制代码
- 复制后显示"已复制"提示

**示例**:
当 AI 回复包含代码时：
```javascript
function hello() {
  console.log("Hello World");
}
```
代码块会有：
- 深色背景
- 圆角边框
- 顶部工具栏（语言 + 复制按钮）

## 📊 改进的表格渲染

**特性**:
- 圆角边框设计
- 表头深色背景突出显示
- 鼠标悬停行高亮
- 表格过宽时支持横向滚动

**示例效果**:
| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 编辑 | 点击铅笔 | 修改已发送消息 |
| 重新生成 | 点击按钮 | 重新请求 AI 回答 |
| 复制代码 | 点击按钮 | 一键复制代码块 |

## 💡 使用技巧

### 编辑消息的最佳实践
1. 编辑后的对话会形成新的分支
2. 原有的后续对话会被删除
3. 适合在对话初期调整方向

### 重新生成的使用时机
- AI 回答太简短
- 需要更详细的解释
- 想要不同的解决方案
- 回答偏离了问题核心

### 代码使用建议
- 复制前检查代码语言是否正确
- 注意 AI 生成的代码需要测试验证
- 可以要求 AI 解释代码细节

## 🎨 界面细节优化

### 会话列表
- 日期格式：简洁的"月日"格式
- 信息布局：标题 + 日期·模型名称

### 消息气泡
- 用户消息：右对齐浅色背景
- AI 消息：左对齐，带头像和模型标识
- 悬停效果：显示操作按钮

### 按钮交互
- 编辑按钮：铅笔图标，悬停显示
- 重新生成：循环箭头图标 + 文字
- 复制代码：复制图标，点击后变为勾号

## 🔧 技术说明

### 浏览器要求
- 需要支持 `navigator.clipboard` API
- 推荐使用现代浏览器（Chrome、Firefox、Safari、Edge 最新版）

### 响应式设计
- 移动端也支持所有功能
- 触摸设备长按显示编辑按钮
- 表格在小屏幕上可横向滚动

## 📝 注意事项

1. **编辑消息会重置对话**: 编辑后的消息之后的所有内容都会被删除
2. **流式生成时禁用**: 当 AI 正在回复时，编辑和重新生成按钮会被禁用
3. **代码语言识别**: 依赖 AI 回复中的代码块语言标记（如 \`\`\`javascript）
4. **网络连接**: 重新生成需要网络连接良好

## 🎯 快速上手

1. **发起对话** → 输入消息，选择模型
2. **查看历史** → 左侧边栏，带日期显示
3. **修正提问** → 悬停消息 → 点击编辑
4. **优化回答** → 点击"重新生成"按钮
5. **使用代码** → 点击代码块的"复制"按钮
